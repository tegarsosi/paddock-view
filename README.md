# paddock-view 
A fullstack Formula 1 data app using FastF1. Built to showcase clean architecture, Go backend, and Next.js frontend with PostgreSQL and Redis.

## Features
- Driver info, race results, standings, and schedule
- API caching with Redis
- Webhook-triggered data pipeline with FastF1
- Dockerized services for easy development

## Tech Stack
- Frontend: Next.js (Vercel)
- Backend: Go (Railway)
- Data layer: Python + FastF1
- Cache: Redis
- DB: PostgreSQL
- Monitoring: Promotheus + Grafana
